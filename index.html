🔵 FRP Checker Pro
App para verificar en tiempo real si un celular tiene solución de desbloqueo FRP disponible con tus herramientas actuales.

Ideal para compra y venta de celulares: consultá un modelo antes de comprarlo para saber si tiene solución o no.

Android: Abrí en Chrome → menú ⋮ → "Agregar a pantalla de inicio"

Windows: Abrí en Chrome → ícono ⊕ en la barra de direcciones → "Instalar FRP Checker Pro"

☁️ Historial en la nube (Supabase)
Creá un proyecto gratuito en supabase.com, ejecutá este SQL en el editor y pegá los datos acá. El historial se sincronizará entre todos tus dispositivos.
CREATE TABLE searches ( id UUID DEFAULT gen_random_uuid() PRIMARY KEY, device_id TEXT, query TEXT NOT NULL, result TEXT, created_at TIMESTAMPTZ DEFAULT NOW() ); ALTER TABLE searches DISABLE ROW LEVEL SECURITY;
